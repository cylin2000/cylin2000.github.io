AngularJS 使用整理

基本概念

  • scope - view与contorller的纽带
  • controller - app的行为(application behavior), controller不应该包含任何显示信息的逻辑
  • model - app的数据
  • view - 用户所看到的东东 //View中不应该包含任何行为
  • directives - HTML的语法扩展
  • filters - 根据用户的本地格式，格式化数据
  • injector - 加载我们的app（依赖管理之类）
  • module - 配置injector
  • $ - angular的命名空间（namespace）
